# FreeFileSync 介绍与使用说明

## 什么是 FreeFileSync

FreeFileSync 是一款免费、开源的文件同步工具，可在 Windows、macOS 和 Linux 平台上使用。它提供了直观的界面，帮助用户在不同位置之间同步文件和文件夹，支持多种同步模式和高级选项。

## 主要功能

- 双向同步：自动检测两边的更改并进行相应同步
- 镜像同步：将源文件夹的内容完全复制到目标文件夹
- 更新同步：只复制新文件或修改过的文件
- 实时同步：监控文件变化并自动同步
- 版本控制：保留文件的历史版本
- 批量重命名：支持基于规则的文件重命名
- 过滤功能：可根据文件类型、大小等条件过滤文件
- 命令行支持：可通过命令行执行同步任务

## 安装方法

1. 访问 [FreeFileSync 官方网站](https://freefilesync.org/)
2. 下载适合您操作系统的安装包
3. 运行安装程序并按照提示完成安装
4. 安装完成后，启动 FreeFileSync

## 基本使用流程

### 1. 创建同步任务

1. 打开 FreeFileSync
2. 点击左侧的"浏览"按钮，选择源文件夹
3. 点击右侧的"浏览"按钮，选择目标文件夹
4. 选择同步模式（双向、镜像、更新）
5. 点击"比较"按钮，查看文件差异

### 2. 配置同步选项

1. 点击"配置"按钮，打开配置对话框
2. 根据需要调整以下选项：
   - 过滤设置：添加文件/文件夹过滤器
   - 冲突处理：设置文件冲突时的处理方式
   - 版本控制：启用并配置版本控制
   - 性能设置：调整同步速度和缓存

### 3. 执行同步

1. 查看比较结果，确认要同步的文件
2. 点击"同步"按钮开始同步过程
3. 同步完成后，查看同步报告

## 高级功能

### 1. 批处理作业

1. 点击"文件" > "保存批处理作业"
2. 为作业命名并保存
3. 可通过命令行或计划任务执行批处理作业

### 2. 实时同步

1. 点击"实时同步"按钮
2. 配置同步间隔和其他选项
3. 启动实时同步，FreeFileSync 将监控文件变化并自动同步

### 3. 文件夹比较

1. 点击"比较"按钮后，可查看详细的文件差异
2. 使用颜色编码区分不同状态的文件：
   - 绿色：仅在源文件夹中存在
   - 蓝色：仅在目标文件夹中存在
   - 黄色：两边都存在但内容不同
   - 灰色：两边都存在且内容相同

## 注意事项

1. **备份重要数据**：在执行同步操作前，建议先备份重要数据
2. **测试同步**：首次使用时，建议先在测试文件夹上进行同步，确保配置正确
3. **定期更新**：保持 FreeFileSync 为最新版本，以获取最新功能和安全修复
4. **合理设置过滤器**：使用过滤器排除不需要同步的文件，提高同步效率
5. **注意文件权限**：确保对源文件夹和目标文件夹有足够的访问权限

## 常见问题

### 1. 同步速度慢怎么办？

- 检查网络连接速度
- 调整性能设置，增加缓存大小
- 减少同步文件数量，使用过滤器排除不需要的文件

### 2. 同步过程中出现错误怎么办？

- 检查文件权限
- 确保文件没有被其他程序占用
- 查看错误日志，了解具体错误原因

### 3. 如何设置自动同步？

- 使用批处理作业和操作系统的计划任务功能
- 或使用实时同步功能，设置适当的同步间隔
